# Smart Password Studio

## Features
- Generate secure passwords
- Adjustable password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Password strength indicator
- Copy password functionality

## Technologies Used
- HTML5
- CSS3
- JavaScript
